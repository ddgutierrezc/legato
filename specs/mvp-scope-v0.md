# MVP Scope v0

## Included
- Monorepo base structure for multi-framework delivery.
- Contract package with canonical domain types/constants.
- Architecture/spec artifacts capturing agreed direction.

## Excluded
- Native playback implementation.
- Framework runtime bindings.
- CI/CD and release automation.
- Advanced features (DRM, offline cache, DSP/effects).

## Success Criteria
1. Repo structure is ready for parallel native/binding work.
2. Contract package acts as source-of-truth.
3. Team has concise specs to guide subsequent phases.
